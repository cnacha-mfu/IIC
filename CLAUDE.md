# CLAUDE.md

This file provides guidance to Claude Code (claude.ai/code) when working with code in this repository.

## Project Overview

This repository contains strategic planning documents for the **Intelligence Innovation Hub (IIH)** — an AI and Robotics Innovation Center proposed at **Mae Fah Luang University (MFU)** in Chiang Rai, Thailand. The project positions MFU as the Greater Mekong Subregion (GMS) gateway for Chinese AI and robotics technology entering the ASEAN market.

Budget: 200 million THB over 3 years (FY 2570-2572 / 2027-2029).

## Repository Structure

All content is in the repository root — no subdirectories (besides `.claude/` and `.git/`).

### Key Documents

- **`blueprint.md`** — Comprehensive strategic blueprint covering global AI center models, humanoid robotics opportunities, Chinese partnerships, funding models, and phased implementation
- **`proposal.md`** — Full Thai-language government funding proposal (IIH) aligned with national strategies (Thailand 4.0, National AI Strategy, GMS cooperation)
- **`proposal_mdes.md`** — Thai-language proposal formatted per MDES (Ministry of Digital Economy) requirements for projects exceeding 200M THB
- **`proposal_old.md`** — Earlier version of the proposal
- **`executive_summary_IIH.md`** — Thai-language executive summary with key metrics and targets
- **`collaboration.md`** — English-language executive summary and action plan identifying 20 target Chinese companies across 4 sectors (robotics, healthcare, agriculture, industrial automation)
- **`collaboration_th.md`** — Thai translation of collaboration.md
- **`event.md`** — Event strategy for the "Mekong AI Forum" — a flagship annual summit

### Supporting Files

- **`infographic_IIH.html`** / **`infographic_IIH.png`** — Visual infographic for the IIH proposal
- **`generate_infographic.py`** — Python script using `html2image` to render the HTML infographic to PNG (hardcoded Google Drive paths)
- **`*.docx`** — Equipment specifications and budget documents (untracked)

## Working with This Repository

- Documents are in **Markdown** with bilingual content: Thai (primary audience: Thai government) and English (strategy/collaboration documents)
- Thai-language documents use formal government proposal style (ราชาศัพท์/ภาษาราชการ)
- Financial figures use Thai fiscal years (พ.ศ. = CE + 543)
- The infographic script requires `html2image` (`pip install html2image`) and a Chromium browser

## Key Context

- **Three strategic pillars**: AI Research & Innovation Hub, Humanoid Robotics Center, Startup & Training Ecosystem
- **Four target sectors for Chinese partnerships**: Humanoid/Service Robotics, Healthcare Robotics, Agriculture AI, Industrial Automation
- **Geographic advantage**: Chiang Rai sits on the R3A Highway corridor to Yunnan, with rail (China-Laos railway) and road connectivity to the GMS
- **Benchmark model**: AI Singapore (AISG) four-pillar structure (research, industry, talent, ecosystem)
